BDD

Les base de donnéessont des magasins qui permettent de sticker des informations et permettent notamment d'effectuer différents opperations de lecture et d'écriture.

Elles sont efficaces pour retrouver des informatios qui sont stockée. Vous pouvez untrroger une base de données, par exemple : "combien d'uttilisateurs sont inscricts sur mon site web?" ou "Quel article dans mon blog contien tel ou tel mot-clé?"

La force des base de données réside également dans leur possibilitées de lier les différents informations qui y sont préalablement stockées.

SGBD

SGBD (Système de Gestion de Base de Données) permetd'acceuilir, d'exploiter et de faire fonctionner les bases de données (moteur). Nous uttiliserons MySQL, d'autre SGBD existe : Oracles, Sql Server, PostgreSQL,etc

MySQL

MySQL (Système de Gestion de Base de Données) c'est l'un des systèmes le plus populaires au mondeet est très utilisé dans le monde du web

Comme sonnom l'indique MySQL uttilise le langage SQL pour effectuer des requettes

Toutes ces actions de gestion et manipulatins passeront par une requête SQL

SQL

SQL(Structured Query Language) langage de requête structurer : est un language informatique permettant d'unteragire avec des base de données relationelles

Une fois que les informations on été enregistrées , il est important de pouvoir les gérer : ajout, modificatons, suppressions , consultations.

phpMyAdmin

phpMyAdmin(Souvent abrégé PMA) est une application web de géstion pour MySQL. IL est écrit en PHP et permet de gérer les BDD dans une interface visuèlle simple et intuitive.

Il est important de notter que PMA n'est q'une application .il est tout à fait possible de gérer MySQL avec d'autre applications similaires ou en ligne de commade

CRUD

CRUD(Create, Read, Ubdate, Delete) désigne les opérations de base pour le stockage et la manipulation d'iformations dans une BDD

  • Create : Insérer des nouveux enregistrements (écriture)
  • Read : Recupérer des enregistrements (lecture)
  • Ubdate : Mettre à jour des enregistrements (écriture)
  • Delete : Supprimer des enregistrements (écriture)

Les opérations correspondent chaqun à un type de requêtes SQL
